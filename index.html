Основы JS: переменные и данные

Программы в Javascript состоят из команд. В конце команды мы ставим ; , чтобы компьютер точно знал, где заканчивается команда.
Например, существует метод console.log() , который используется для вывода сообщений в консоль.
Для хранения данных используются переменные :

  • Let - ключевое слово для обозначения переменной, которую можно изменить
    let age = 37;
    age = 38;
    console.log(age); // мы увидим 38
  • Const - константа, то есть ключевое слово для переменной, которую нельзя менять
    const name = 'Misha';
    name = 'Petya'; // будет ошибка

Обычно программы работают с данными . Это могут быть, например, какие-то числа или текст. Для записи данных в программе используются литералы .

  1. Числа состоят из цифр и пишутся без кавычек (42, -5, 1000000, 12.67, -0.2). JS str 43.
  2. Строки пишутся в кавычках ('Misha', 'Hello, world!'). JS str 56.
  3. Boolean - Булев (логический) тип состоит из 2 значений true (правда) и false (ложь). Такие значения используются для проверки условий в конструкции if и циклах. JS str 77.

Оператор typeof позволяет узнать тип данных переменной или целого выражения. JS str 30. IT IS COOL
